# Machine Learning Performance Prediction Script

This repository contains the Python-based machine learning pipeline developed for the Master's thesis:

**AI-based Analysis and Prediction of the Performance of Angular and React Applications**

## Purpose

The script analyses collected benchmark data and predicts frontend performance metrics based on structural application features.

## Input Features

- Framework
- Items
- Scenario
- Lines of Code
- Component Count
- Component Depth

## Target Variables

- CPU Time
- JavaScript Execution Time
- Memory Usage

## Implemented Models

- Linear Regression
- Random Forest Regression

## Evaluation Metrics

- R² Score
- RMSE

## Tech Stack

- Python
- pandas
- scikit-learn
- numpy
